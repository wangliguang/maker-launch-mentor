# 硬件、工具与耗材信息

本文件保存硬件、工具和耗材的通用资料，不表示现场拥有这些物品，也不维护库存状态。具体项目需要什么以及如何使用，由对应的 `PROJECT.md` 和主 `SKILL.md` 决定。

## 硬件信息入库规则

入库只表示增加一种硬件的信息，不表示增加实物库存。

1. 每种硬件需要记录名称、通用用途和注意事项。
2. 不在本文件绑定具体项目、玩法、代码文件或固定 GPIO 引脚。
3. 工具和耗材单独列出。
4. 不记录数量、剩余数量、借出状态、可用状态或某位学员是否拥有。

## 硬件

| 名称 | 通用用途 | 注意事项 |
| ---- | -------- | -------- |
| ESP32 开发板 | 运行程序，读取传感器和按钮，控制显示、灯光、蜂鸣器等外设。 | 接线前先断电；确认供电电压和引脚电压符合所连接模块的要求。 |
| TM1637 四位数码管模块 | 显示数字、倒计时、分数和简单状态码。 | 接线时核对电源正负极及模块上的 CLK、DIO 标识，具体引脚由项目说明决定。 |
| 按钮 | 提供开始、确认、选择、抢答或复位等人工输入。 | 具体接线和按下判定由项目说明决定；引导学员时只给具体接线，不让学员选择上拉或下拉。 |
| LED 指示灯 | 显示状态、结果、提醒或调试信息。 | LED 有正负极，接反不会正常发光；使用时通常需要串联合适的限流电阻。 |
| 220 欧姆电阻 | 为 LED 等元件提供限流。 | 电阻没有方向；使用前确认阻值适合所连接的元件和供电电压。 |
| 洞洞板 | 固定电子元件并形成稳定电路。 | 焊接前规划元件位置和连接路径；加工或焊接时遵守对应工具的安全要求。 |
| M2 主板隔离柱 | 固定小型模块，并让模块和底板保持间距。 | 使用前确认螺纹规格、长度和安装孔尺寸匹配，避免压到板上元件。 |
| M2 尼龙螺丝 | 固定小型模块或电路板。 | 使用前确认长度和安装孔尺寸匹配，不要过度拧紧以免损坏电路板。 |
| M2 尼龙螺母 | 配合 M2 螺丝固定模块或电路板。 | 与 M2 螺丝配套使用，拧紧时避免让电路板弯曲。 |
| 纯铜双头镀锡导线 | 连接洞洞板、电路模块和电子元件。 | 连接前断电并确认两端位置；裸露导体不得互相接触造成短路。 |
| USB 数据线 | 为开发板供电、传输程序和提供串口通信。 | 确认线材支持数据传输且接口匹配；插拔时握住插头，不要拉扯线缆。 |

## 工具与耗材

| 类别 | 名称 | 通用用途 | 注意事项 |
| ---- | ---- | -------- | -------- |
| 加工工具 | 小型打磨机 | 修整洞洞板、外壳或安装孔位。 | 可能造成割伤或飞溅，只能由家长到场并全程陪同、手把手指导使用。 |
| 焊接工具 | 电烙铁 | 焊接导线、元件、模块和洞洞板连接点。 | 高温可能造成烫伤，只能由家长到场并全程陪同、手把手指导使用。 |
| 焊接耗材 | 焊锡 | 固定导线和元件脚，形成电气连接。 | 焊接时保持通风，避免接触高温焊锡，并在家长全程陪同下使用。 |
| 辅助耗材 | 基础焊接用品（未细分） | 辅助完成焊接。 | 使用前根据具体物品确认用途和安全要求；涉及焊接时必须由家长全程陪同。 |
| 调试工具 | Arduino IDE 或等效上传环境 | 编写、编译和上传 Arduino 程序，查看串口信息。 | 上传前确认开发板型号和端口设置正确，避免在接线带电变更时操作。 |
