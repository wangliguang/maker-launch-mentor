# 硬件清单及工具

选择、新增或修改硬件前，先读这个文件。

## 硬件入库规则

1. 每个硬件需要写清楚名称和通用用途。
2. 不在本文件绑定具体项目、玩法、代码文件或固定 GPIO 引脚。
3. 如果硬件有方向、安全限制或常见注意事项，需要写在备注里。
4. 工具和耗材单独列出，方便制作前检查。

## 硬件清单

| 名称 | 通用用途 | 注意事项 |
| ---- | -------- | -------- |
| ESP32 开发板 | 作为 Arduino/物联网项目的主控板，可读取传感器和按钮，也可控制显示、灯光、蜂鸣器等外设。 | 使用前确认开发板型号、供电方式、USB 端口和 Arduino IDE 开发板配置。 |
| TM1637 四位数码管模块 | 显示数字、倒计时、分数、状态码等简单信息。 | 需要连接电源、GND、CLK 和 DIO；实际引脚由项目代码决定。 |
| 按钮 | 作为人工输入开关，可用于开始、确认、选择、抢答、复位等操作。 | 常见接法是配合上拉或下拉电阻；使用内部上拉时，按下通常接 GND。 |
| LED 指示灯 | 显示状态、结果、提醒或调试信息。 | 必须串联限流电阻；注意 LED 正负极方向。 |
| 220 欧姆电阻 | 常用于 LED 限流，保护 LED 和主控板引脚。 | 每个普通 LED 通常需要单独串联一个限流电阻。 |
| 洞洞板 | 固定电子元件，并通过焊接形成稳定电路。 | 焊接前先规划元件位置，避免后续返工。 |
| M2 主板隔离柱 | 固定小型模块或电路板，并让模块和底板保持间距。 | 需要配合合适尺寸的螺丝和螺母使用。 |
| M2 尼龙螺丝 | 固定小型模块或电路板。 | 尼龙材质不易导电，适合固定电子模块。 |
| M2 尼龙螺母 | 配合 M2 螺丝固定模块或电路板。 | 拧紧即可，不要过度用力。 |
| 纯铜双头镀锡导线 | 用于洞洞板、电路模块和电子元件之间的连接。 | 镀锡端方便焊接，焊接后检查是否牢固。 |
| USB 数据线 | 给开发板供电、上传程序或进行串口调试。 | 需要支持数据传输，不能只支持充电。 |

## 工具与耗材

| 类别 | 名称 | 用途 | 注意事项 |
| ---- | ---- | ---- | -------- |
| 加工工具 | 小型打磨机 | 修整洞洞板、外壳或安装孔位。 | 使用时注意固定材料，避免打磨过度。 |
| 焊接工具 | 电烙铁 | 焊接导线、按钮、LED、电阻、模块和洞洞板连接点。 | 焊接前确认元件位置和引脚，避免反复拆焊。 |
| 焊接耗材 | 焊锡 | 固定导线和元件脚，形成可靠电气连接。 | 焊点应光亮、牢固，避免连锡和虚焊。 |
| 调试工具 | Arduino IDE 或等效上传环境 | 编写、编译和上传 Arduino 程序，查看串口调试信息。 | 串口波特率需要和代码中的设置一致。 |

## 通用制作检查

1. 主控板和外设模块需要共地。
2. LED 使用前先确认正负极，并串联限流电阻。
3. 按钮接线前先确认项目使用上拉还是下拉逻辑。
4. 显示模块接线前先确认电源、GND、数据线和时钟线。
5. 焊接完成后检查是否有连锡、虚焊或松动导线。
6. 上传代码前确认 USB 数据线支持数据传输。
